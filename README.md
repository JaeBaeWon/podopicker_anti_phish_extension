# Podopicker Anti-Phish Extension

> 티켓팅 사이트 피싱 공격으로부터 사용자를 보호하는 브라우저 확장 프로그램

## 🎯 개요

Podopicker Anti-Phish Extension은 티켓팅 사이트를 모방한 피싱 사이트로부터 사용자를 보호하기 위해 개발된 브라우저 확장 프로그램입니다. 실시간으로 URL을 검사하여 피싱 도메인 접속을 차단하고, 사용자에게 안전한 접속 방법을 안내합니다.

## 🚨 배경

### 문제 상황
- 티켓팅 사이트에서 피싱 사이트로 유도되는 사례가 빈번하게 발생
- 기존 해결책: 공식 오픈 알림 이메일 발송을 통한 안전한 접속 유도

### 피드백 및 개선점
**피드백**: "피싱 메일도 많은데요?"
- 피싱 공격자가 동일한 이메일 형식을 모방할 위험
- 이메일 자체가 또 다른 공격 경로가 될 수 있음

**해결책**: 브라우저 확장 프로그램을 통한 클라이언트 단 보호

## ✨ 주요 기능

### 🔍 URL 검사
- 실시간으로 접근하려는 URL 분석
- 피싱 도메인 패턴 탐지 및 차단

### 🛡️ 피싱 탐지 알고리즘
다음과 같은 유사 문자 변환을 통한 피싱 도메인 탐지:
- `o` ↔ `0` (영문 소문자 o와 숫자 0)
- `rn` ↔ `m` (rn 조합과 m)
- `l` ↔ `1` (영문 소문자 l과 숫자 1)

### 🚫 차단 및 안내
- 피싱 도메인 감지 시 즉시 `block.html`로 리디렉션
- 차단 이유 및 안전한 접속 방법 안내
- Podopicker 브랜드 디자인 적용

## ⚙️ 동작 원리

1. **URL 인터셉트**: Background/Content Script가 브라우저 요청을 모니터링
2. **패턴 분석**: 접근하려는 URL을 피싱 도메인 데이터베이스와 비교
3. **차단 실행**: 피싱 도메인으로 판단 시 `block.html`로 리디렉션
4. **사용자 안내**: 차단 이유 및 안전한 대안 제시

### 구성 요소
- **Background/Content Script**: URL 인터셉트 및 검사
- **Block Page (block.html)**: 차단 안내 페이지
- **Manifest.json**: 확장 프로그램 설정 및 권한 정의

## 🚀 설치 및 사용법

### 개발자 모드 설치
1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 프로젝트 폴더 선택

### 사용법
1. 확장 프로그램 설치 후 자동으로 활성화
2. 피싱 의심 사이트 접속 시 자동으로 차단 페이지 표시
3. 차단 페이지에서 안전한 공식 사이트로 이동 가능

## 🔮 향후 개선 사항
